# App Store & Google Play Screenshots Strategy

This folder contains the metadata needed for store listings. For actual screenshot assets, you should generate them using the following plan:

## Screenshot Requirements
- **App Store**: 6.5 inch (1284 x 2778) and 5.5 inch (1242 x 2208)
- **Google Play**: Phone (1080 x 1920 or similar 16:9/19.5:9 aspects)

## Recommended Screenshot Flow
1. **Hero / Home Screen**: 
   - *Caption*: "Your Ultimate Streaming Guide"
   - *Visual*: The Home Screen showing personalized "Discovery" and "Trending" carousels.
2. **Detail Screen / Where to Watch**:
   - *Caption*: "Know Exactly Where to Watch"
   - *Visual*: A movie detail screen highlighting the Streaming Services row (Netflix, Prime, etc.).
3. **Watchlist / Tracker**:
   - *Caption*: "Never Lose Track Again"
   - *Visual*: The Watchlist screen showing saved movies and TV shows.
4. **AI Recommendations**:
   - *Caption*: "AI-Powered Suggestions"
   - *Visual*: The AI Chat screen showing a personalized recommendation conversation.
5. **Statistics / Year in Review**:
   - *Caption*: "Visualize Your Taste"
   - *Visual*: The Statistics dashboard with charts and top genres.

*Note: Use tools like **Fastlane Frameit**, **Rotato**, or **AppMockUp** to place raw app screenshots into device frames and add the captions.*
